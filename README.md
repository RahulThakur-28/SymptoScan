# 🩺 SymptoScan - AI Health Checker

SymptoScan is an Android app that helps users analyze symptoms using AI and get possible health insights instantly.

---

## 🚀 Features

* 🤖 AI-powered symptom analysis (Gemini API)
* 💬 Chat with AI assistant
* 📊 Risk level detection (Low / Medium / High)
* 🕒 History tracking (Room DB + Firebase)
* 🔐 User Authentication (Firebase)
* 🌙 Dark Mode support
* 🎨 Modern UI with Jetpack Compose

## ⚙️ Setup Instructions

1. Clone repo:

```
git clone https://github.com/RahulThakur-28/SymptoScan.git
```

2. Open in Android Studio

3. Add your API key in repository:

``
Use local.properties to store your API key securely.
```

4. Run the app 🚀
